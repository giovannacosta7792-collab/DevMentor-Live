# DevMentor-Live
Aplicativo de videoconferência para mentoria de desenvolvedores com integração do Jitsi Meet e suporte de IA para análise de código.

# 📱 DevMentor Live

## 🎯 Proposta de Valor

O DevMentor Live é um aplicativo de videoconferência focado em mentoria para desenvolvedores, permitindo chamadas em tempo real com suporte de Inteligência Artificial para análise de código.

## 🚀 Problema Resolvido

Desenvolvedores iniciantes têm dificuldade em explicar problemas técnicos e receber feedback estruturado. O app facilita esse processo com videochamadas e suporte de IA.

## 🛠 Tecnologias Utilizadas

* Kotlin (Android)
* Jitsi Meet SDK
* Manus AI
* Android Studio

## 📲 Funcionalidades

* Criar/entrar em salas de videochamada
* Compartilhar código
* Análise automática com IA

## ▶️ Como Instalar

1. Clone o repositório:
   git clone https://github.com/seuusuario/devmentor-live

2. Abra no Android Studio

3. Execute em um emulador ou celular
